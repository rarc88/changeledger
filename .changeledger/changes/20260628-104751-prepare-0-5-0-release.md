## Request

Dejar la evolución de contexto dinámico lista para publicar como ChangeLedger
`0.5.0` y usarla durante la próxima semana en proyectos reales. El plan de
release vigente calcula `0.4.0 → 0.5.0` con impacto `minor` e incluye los changes
`20260627-205033`, `20260627-205034`, `20260627-212133` y
`20260627-215619`.

La revisión previa encontró residuos editoriales del contrato monolítico en el
README, la configuración generada, mensajes del CLI y comentarios del workflow
de publicación, además de pequeñas duplicaciones dentro de los fragmentos. La
preparación debe corregirlos y demostrar que el artefacto que recibirán los
usuarios contiene y ejecuta el nuevo compilador de contexto.

Este change deja el repositorio **listo para publicar**, pero no crea tags, no
publica una GitHub Release, no hace push y no ejecuta `npm publish`; esas acciones
externas se harán después de la validación humana.
